\doxysubsection{Detailed Description}
A UI elemnt that represent a drop down menu. 

A class derived from \mbox{\hyperlink{class_button}{Button}} class. It contains a list of menus inside a single element. The list of items are left aligned and below the menu title. the size of the items is depended on the longest item name.\hypertarget{class_drop_down_menu_autotoc_md2}{}\doxyparagraph{Example Case}\label{class_drop_down_menu_autotoc_md2}
