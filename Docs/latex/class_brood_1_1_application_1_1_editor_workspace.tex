Initializes the edit mode tab bar which is at the top of the edit window.

It has the 4 edit modes in it\+: \mbox{\hyperlink{class_board}{Board}} Edit, \mbox{\hyperlink{class_tiles}{Tiles}} Edit, \mbox{\hyperlink{class_path}{Path}} Edit, and Deck Edit

\mbox{\hyperlink{class_board}{Board}} Edit allows for the general settings to be edited.

Tile Edit allows for tiles setting to be changed for indivisual tiles.

\mbox{\hyperlink{class_path}{Path}} Edit allows for the path that the players take to be changed.
