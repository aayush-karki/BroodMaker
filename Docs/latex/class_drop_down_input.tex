\doxysubsection{Detailed Description}
A UI elemnt that represent a drop down menu. 

A class derived from \mbox{\hyperlink{class_drop_down_menu}{Drop\+Down\+Menu}} class. It contains a list of menus inside a single element. The list of items are left aligned and below the menu title. the size of the items is depended on the longest item name.

\begin{DoxyNote}{Note}
This class differes from the \mbox{\hyperlink{class_drop_down_menu}{Drop\+Down\+Menu}} class as for this class the title that is used in \mbox{\hyperlink{class_drop_down_menu}{Drop\+Down\+Menu}} is changed to relfect the selected item. So it is important to add line of code below when your item\textquotesingle{}s Do\+Element() returns true.
