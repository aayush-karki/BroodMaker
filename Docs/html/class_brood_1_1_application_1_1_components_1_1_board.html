◆ UpdateBoardPath()

void Board::UpdateBoardPath ( unsigned  a_rowBegin,
unsigned  a_rowEnd,
unsigned  a_colBegin,
unsigned  a_colEnd,
bool  a_createNew = false,
Brood::Application::Components::Deck *  a_deckPtr = nullptr 
)
private

updates the path specified by passed param in board by using the member variables.

It updates the tile starting form a_rowBegin to including a_rowEnd for the rows and for the column starting from a_colBegin to including a_colEnd. It calls Brood::Application::Components::Tiles::UpdateTile() on each tile.

Parameters
a_rowBeginindex of row from where the tiles update should start
a_rowEndindex of row from where the tiles update should end
a_colBeginindex of column from where the tiles update should start
a_colEndindex of column from where the tiles update should end
a_createNewtrue if new tiles should be created before updating the tiles -> default false
