void SpriteHandler::SetSpriteFromTexture ( unsigned  a_num)

Setter for texture.

sets the texture from the texture vector if it is not empty. If either the sprite height and length is equal to zero then, it sets them to body's height and length

Warning
Assumes that The sprite for the faces are lined linearly. That is assumes that there is only one row of sprite.
