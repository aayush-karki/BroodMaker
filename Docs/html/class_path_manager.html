Detailed Description

A PathManager Class that manages the linear flow of how a player moves.

It converts the 2d board into a 1d line that makes it easer to compute the path that a player would follow

Note
By default it creats a snake like path. say, if the board is of 3,3 say 0,0 | 0,1 | 0,2 1,0 | 1,1 | 1,2 2,0 | 2,1 | 2,2

Then the default path would be: 0,0 -> 0,1 -> 0,2 | v 1,0 <- 1,1 <- 1,2 | v 2,0 -> 2,1 -> 2,2

or, in a list: 

0,0 -> 0,1 -> 0,2 -> 1,2 -> 1,1 -> 1,0 -> 2,0 -> 2,1 -> 2,2
